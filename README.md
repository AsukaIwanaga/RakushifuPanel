# らくしふ 客数予測パネル (v2)

らくしふのシフト編集画面 (`/admin/v2/schedules`) に、スプレッドシート
「時間帯別客数予測 v2」の当日データを重ねて表示するChrome拡張。

**v2から店舗固有の値（シートID・genre分類・正社員名）は設定画面で入力する方式になり、
他店舗でもコード変更なしで使えます。**

## 表示内容

- **上部チップ**: LABOR% / LABOR H / SALES / SBP（シートヘッダー部）
- **時間帯テーブル** (6:00–23:00): LE(客数) / REQ F / REQ K / REQ計
  ＋ らくしふ実シフトから集計した実F / 実K / 実計（休憩控除済み）と不足行
  - 今日を表示中は現在時刻の列をハイライト、不足時間帯は赤表示
- **タスク**: タスクシートから対象日に該当する月次/週次/要請タスクを表示
  （タスクシート未設定なら非表示）
- **シフト確定 未処理日**: 今日〜月末で「シフト確定」がまだの日を赤チップで列挙。
  📊ボタン（画面右上）にも件数バッジ表示。5分ごとに自動更新。

対象日はらくしふURLの `from=` に自動追従。パネルの ◀ ▶ で独立して前後の日も見られる。

## インストール（ストア不要）

1. このリポジトリを取得して展開
   - `git clone https://github.com/AsukaIwanaga/RakushifuPanel.git`（`v2` ブランチ）
   - または GitHubページの **Code → Download ZIP**
2. Chromeで `chrome://extensions` を開く
3. 右上「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」→ 展開したフォルダを選択

## 初期設定（店舗ごとに必要）

`chrome://extensions` → 本拡張の「詳細」→「拡張機能のオプション」を開き
（らくしふ画面のパネルに出る「設定を開く」ボタンからも開けます）:

1. **客数予測シート**（必須）: 自店舗の「時間帯別客数予測 v2」のURLを貼り付け
   - 日付タブ名が `MMDD (曜)` 形式（例: `0718 (土)`）であること
   - シートは自分のGoogleアカウントで閲覧できる状態にしておく
2. **タスクシート**（任意）: タスク定義シートのURL。使わなければ空欄
3. **genre_id**: フロア/キッチンのgenre_id。らくしふ編集画面でグループ絞り込みを
   フロアのみ/キッチンのみにしたときURLに付く `g=` の数字（例: `g=2` → 2）
4. **正社員の氏名**（任意）: MGT/TRer/TRee時間の計上先（MGT/cMGT）の振り分けに使用

保存したら、らくしふのページを再読み込み → 右上に📊ボタンが出ます。

## 仕組み

- シート取得: `docs.google.com` の gviz CSVエンドポイントを、ブラウザのGoogleログイン
  Cookieで取得（APIキー不要・シートの閲覧権限があれば動く）
- 行の特定はシートG列のラベル (`LE`, `REQ（F）` など)。表示項目を変えたいときは
  `content.js` 冒頭の `HOURLY_COLS` / `HEADER_LABELS` を編集
- 実人数: らくしふ内部API `/ajax/admin/v2/schedules` から対象日のシフトを取得し、
  休憩控除・業務割振（F/K/FK振替、BU系→K、MGT/TRer/TRee→MGT系）込みで時間帯別に集計
- 未確定日: らくしふ内部API `shift_confirm_target_candidates` を利用。
  **レスポンス形式は防御的にパースしているため、初回は表示が実態と合っているか要確認。**

## 既知の注意点

- Googleからログアウトすると「シート取得失敗」になる → Googleに再ログイン
- シートの列配置（G列ラベル・6:00開始・J列ヘッダー値）はテンプレート準拠が前提。
  変えるとパースが壊れる（その場合は `content.js` の `COL_*` 定数を調整）
- 業務割振タスク名（F/K/FK/BU*/MGT/TRer/TRee）の解釈は共通ルールとしてコード内に固定。
  店舗で名前が違う場合は `content.js` の `moveGroup` を調整
- 設定を変更したら、らくしふページを再読み込み（自動リロードはしない）
- コードを更新したら `chrome://extensions` で ⟳（再読み込み）→ らくしふページもリロード
